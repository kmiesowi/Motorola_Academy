Creation date:

The main requirements from the instruction:


  • The page should contain at least 4 subpages.
  • Its html should be W3C valid.
  • It should contain properly used HTML5 elements.
  • It should contain a navigation menu.
  • It should contain a contact page with a form to send email to the author(without server-side logic).
  • Optional: The contct page should have validation written in JavaScript using regular expressions.
  • Optional: It should contain a welcome screening which displays an animation.

And now some description about my website - appearance and operation!


First of all

- welcome screening with displays an animation. It is two different solutions - Home page and Gallery welcome screenings.


Second of all

- each images are with proper usage rights. I have find commercial use photos and graphics on the PIXABAY website or in email footer (Motorola logo).


Form validation:

It is important to notice validation of the form fields.


Input Name

Not empty and does not contain only whitespace characters. Should contains only letters and space and each word in it starts with capital letters. Should be written at least two words.

Invalid name- red inscription about it.

Form validation:

Input Email

Not empty and does not contain only whitespace characters. Should contains valid email address (aaa@aa.aa).

Invalid email- red inscription about it.


Input Message

Valid if not empty.

Form validation:

Submit Button - SEND

Disabled by default. Enable after valid completion of the form.


After clicking SEND - an alert with written name and information that the message has been sent will be displayed. Then, each field is cleared.
